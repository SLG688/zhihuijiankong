# 智慧健控微信小程序用户使用手册

## 1. 简介

智慧健控是一款专为老弱病残人群设计的微信小程序，提供康复护理和健康保养服务。通过本小程序，您可以方便地管理健康数据、获取康复训练指导、设置用药提醒、在紧急情况下求助，以及学习护理知识。

## 2. 如何使用

### 2.1 打开小程序

1. 打开微信，点击底部的"发现" tab
2. 点击"小程序"进入小程序列表
3. 在搜索框中输入"智慧健控"，点击搜索
4. 点击搜索结果中的"智慧健控"进入小程序

### 2.2 首次使用

1. 首次进入小程序时，会提示您授权获取用户信息
2. 点击"允许"授权，以便小程序为您提供个性化服务
3. 进入首页，开始使用各项功能

## 3. 功能介绍

### 3.1 首页

首页是小程序的入口，包含以下功能：

- **欢迎区域**：显示欢迎信息
- **功能入口**：快速进入健康监测、康复训练、用药提醒、护理知识等功能
- **健康数据概览**：显示血压、心率、血糖等关键健康指标
- **今日提醒**：显示当天的用药和训练提醒
- **紧急求助按钮**：位于右下角，点击可快速进入紧急求助页面

### 3.2 健康档案

健康档案页面用于管理个人健康信息：

- **个人信息**：显示姓名、年龄、性别、联系电话、紧急联系人等信息
- **健康状况**：显示既往病史、过敏史、当前用药等信息
- **健康数据记录**：查看血压、血糖、心率等数据的记录
- **健康报告**：生成和查看健康报告

### 3.3 健康监测

健康监测页面用于记录和管理健康数据：

- **数据记录**：手动录入血压、血糖、心率等数据
- **数据趋势**：查看健康数据的变化趋势
- **健康建议**：获取基于健康数据的建议
- **设备连接**：连接智能健康设备，自动获取数据

### 3.4 康复训练

康复训练页面提供专业的康复训练指导：

- **训练计划**：选择适合的康复训练计划
- **训练指导**：查看训练前准备和注意事项
- **训练记录**：记录训练历史和进度
- **视频指导**：观看训练视频教程

### 3.5 用药提醒

用药提醒页面帮助您管理用药：

- **今日用药**：显示当天需要服用的药物和时间
- **用药管理**：添加、编辑、删除用药提醒
- **用药记录**：查看历史用药记录
- **用药知识**：学习用药相关知识

### 3.6 紧急求助

紧急求助页面在紧急情况下使用：

- **紧急求助按钮**：点击后立即联系紧急联系人
- **紧急联系人**：管理紧急联系人信息
- **附近医院**：查看附近的医院信息并导航
- **急救常识**：学习基本的急救知识

### 3.7 护理知识

护理知识页面提供丰富的健康知识：

- **知识分类**：按分类浏览健康知识
- **知识列表**：查看健康知识文章
- **健康资讯**：了解最新的健康资讯
- **常见问题**：查看常见健康问题的解答

## 4. 操作指南

### 4.1 记录健康数据

1. 进入"健康监测"页面
2. 在相应的数据输入框中输入数据
3. 点击"记录"按钮保存数据
4. 可在"健康档案"页面查看历史记录

### 4.2 设置用药提醒

1. 进入"用药提醒"页面
2. 点击"添加用药提醒"按钮
3. 填写药物名称、剂量、服用时间等信息
4. 保存设置，系统会在指定时间提醒您服药

### 4.3 开始康复训练

1. 进入"康复训练"页面
2. 选择适合的训练计划
3. 点击"开始训练"按钮
4. 按照指导进行训练
5. 训练完成后，系统会记录训练情况

### 4.4 使用紧急求助

1. 在任何页面点击右下角的"紧急求助"按钮
2. 或直接进入"紧急求助"页面
3. 点击"紧急求助"按钮
4. 确认后，系统会联系您的紧急联系人

### 4.5 查看护理知识

1. 进入"护理知识"页面
2. 选择感兴趣的知识分类
3. 点击知识文章查看详情
4. 可在"健康资讯"和"常见问题"部分获取更多信息

## 5. 注意事项

### 5.1 数据安全

- 您的健康数据仅存储在本地和您授权的云端，不会被分享给第三方
- 请妥善保管您的微信账号，避免他人访问您的健康数据

### 5.2 用药安全

- 用药提醒功能仅作为辅助工具，不能替代医生的专业建议
- 请严格按照医生的处方服药，如有疑问请咨询医生

### 5.3 紧急求助

- 紧急求助功能仅在真正紧急情况下使用
- 如遇紧急医疗情况，请立即拨打急救电话

### 5.4 设备连接

- 连接智能健康设备时，请按照设备说明书操作
- 确保设备电量充足，信号良好

## 6. 常见问题

### 6.1 如何修改个人信息？

进入"健康档案"页面，点击"编辑个人信息"按钮，修改相应信息后保存。

### 6.2 如何添加紧急联系人？

进入"紧急求助"页面，点击"添加紧急联系人"按钮，填写联系人信息后保存。

### 6.3 如何查看历史健康数据？

进入"健康档案"页面，点击相应的健康数据记录查看历史数据。

### 6.4 如何调整字体大小？

目前小程序支持默认字体大小，后续版本将添加字体大小调整功能。

### 6.5 如何删除用药提醒？

进入"用药提醒"页面，点击"删除用药提醒"按钮，选择要删除的提醒进行删除。
