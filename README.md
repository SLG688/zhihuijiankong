# 智慧健控 - 面向老弱病残人群的健康医疗小程序

## 项目概述

智慧健控是一款专为老弱病残人群设计的健康医疗小程序，采用微信小程序+Flask+SQLite架构，完全符合微信适老化设计规范。本项目旨在为用户提供便捷的健康监测、用药提醒、康复训练等功能，帮助用户更好地管理个人健康。

## 核心功能

### 1. 健康监测
- 血压记录与分析
- 心率监测
- 血糖记录
- 健康数据可视化

### 2. 用药提醒
- 定时用药提醒
- 用药状态管理
- 药物信息记录
- 用药历史查询

### 3. 康复训练
- 个性化训练计划
- 训练进度跟踪
- 训练指导视频

### 4. 护理知识
- 健康科普文章
- 疾病预防知识
- 护理技巧分享

### 5. 紧急求助
- 一键紧急求助
- 紧急联系人管理
- 附近医院查询
- 急救常识指导

### 6. 健康档案
- 个人基本信息管理
- 健康状况记录
- 就诊历史查询
- 体检报告管理

### 7. 社区功能
- 健康经验分享
- 病友交流
- 专家在线咨询

## 技术栈

### 前端技术
- **微信小程序**：使用原生微信小程序开发框架
- **WXML**：微信小程序标记语言
- **WXSS**：微信小程序样式语言
- **JavaScript**：页面逻辑处理

### 后端技术
- **Flask**：Python Web框架
- **SQLite**：轻量级数据库
- **RESTful API**：前后端数据交互

### 设计规范
- **微信适老化设计规范**：大字体、高对比度、简化操作流程
- **响应式设计**：适配不同屏幕尺寸
- **无障碍设计**：支持语音辅助功能

## 安装与配置

### 前端安装
1. 克隆项目仓库
   ```bash
   git clone https://github.com/SLG688/zhihuijiankong.git
   ```

2. 打开微信开发者工具
3. 导入项目目录
4. 配置AppID（可选）
5. 编译并运行小程序

### 后端安装
1. 进入项目后端目录
   ```bash
   cd backend
   ```

2. 安装依赖
   ```bash
   pip install -r requirements.txt
   ```

3. 初始化数据库
   ```bash
   python init_db.py
   ```

4. 启动后端服务
   ```bash
   python app.py
   ```

## 使用指南

### 首次使用
1. 打开微信小程序
2. 授权登录
3. 填写个人基本信息
4. 设置紧急联系人
5. 完善健康状况

### 日常使用
- **健康监测**：点击"健康监测"入口，记录血压、心率、血糖等数据
- **用药提醒**：在"用药提醒"页面添加药物信息和提醒时间
- **康复训练**：根据个人情况选择合适的训练计划
- **紧急求助**：遇到紧急情况时，点击右下角的"紧急求助"按钮

### 功能操作
1. **添加用药提醒**：
   - 点击"新建提醒"按钮
   - 填写药物名称、剂量、用药时间等信息
   - 保存提醒设置

2. **记录健康数据**：
   - 进入相应的健康监测页面
   - 输入测量数据
   - 点击"保存"按钮

3. **紧急求助**：
   - 点击右下角红色"紧急求助"按钮
   - 系统会自动联系预设的紧急联系人

## API文档

### 健康数据API
- **GET /api/health-data**：获取健康数据列表
- **POST /api/health-data**：添加健康数据
- **PUT /api/health-data/:id**：更新健康数据
- **DELETE /api/health-data/:id**：删除健康数据

### 用药提醒API
- **GET /api/medication**：获取用药提醒列表
- **POST /api/medication**：添加用药提醒
- **PUT /api/medication/:id**：更新用药提醒
- **DELETE /api/medication/:id**：删除用药提醒

### 用户信息API
- **GET /api/user**：获取用户信息
- **PUT /api/user**：更新用户信息
